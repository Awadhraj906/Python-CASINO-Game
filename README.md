# 🎰 APNACASINO

A text-based casino game built in Python where players purchase casino cash and enjoy multiple betting games to win or lose virtual money.

## 🎮 Games Included

### 🏎‍🟀 F1 Racing Bet
- Bet on your favourite Formula 1 driver.
- Dynamic race commentary.
- Random race events like rain and crashes.
- Win or lose casino cash based on your prediction.

### 🔢 Guess the Number
- Guess a random number between 1 and 10.
- Three chances to win.
- Entry fee and rewards based on the game rules.

### 🎲 Luck in the Draw
- Random lucky draw with different outcomes.
- Possible rewards include:
  - Gold Jackpot
  - Lottery Prize
  - Extra Draw
  - Nothing
  - Cash Losses

## 💰 Casino System

- Purchase casino cash before playing.
- Each game has its own entry fee.
- Balance updates after every game.
- Checkout system converts remaining casino cash back into money.

## 🛠️ Built With

- Python 3
- random module
- time module
